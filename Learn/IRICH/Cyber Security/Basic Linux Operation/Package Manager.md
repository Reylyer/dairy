Package manager merupakan sebuah software yang mengatur urusan program atau library dalam sistem operasi. Package manager akan mengatur alur penginstalan, penghapusan, dan mengurus dependencies. Suatu program bisa saja *depends on* beberapa package lain, dan bisa saja 2 program *depends on* program atau library yang sama. Package manager juga mengurus 2 hal itu.

### Contoh package manager di level sistem operasi
- dpkg milik debian based distribution
- apt, sebagai wrapper dpkg 
- pacman milik arch linux
- portage milik gentoo linux
- dnf milik fedora

---

### Contoh package manager sebuah program
- PIP milik python
- NPM milik Node.js
- Cargo milik Rust
- gem milik Ruby
